Welcome to the ummer project page

ummer is a little tool for dealing on the webMethods Universal Messaging (UM) on the command line.

The client libraries of the UM are proprietary. Hence they are not included in the release.

Prerequisites

You do need a Java JRE 8. Get it from here.

Installation

  1. Download the zip
  2. Extract it
  3. Copy from /UniversalMessaging/lib the files
    • nAdminAPI.jar
    • nClient.jar
    • nP2P.jar into the folder /umLib
  4. Done!

Usage

From the command line just hit <path to ummer>/bin/ummer and the usage help list show up.

Show all queues

./ummer -r nsp://127.0.0.1 --listQueues

/de/kazkazi/me
/de/kazkazi/Simple
/de/kazkazi/test

Show all topics

ummer -r nsp://127.0.0.1 --listTopics

/de/kazkazi/aChannel
